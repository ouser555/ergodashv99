## ergodash v99 組裝說明
剛開始拿到套件時可以先清點一下零件
* 鍵盤PCB *2
* 自製Arduino ProMicro *2
* 壓克力 *2組 (左右共6片)
* m2*6mm銅柱 *16
* m2*5mm螺絲 *26
* m2*8mm螺絲 *6
* 可插拔針座
  * 2px1 *4
  * 2px2 *4
  * 3px2 *2 
* 電阻 * 70
* 二極體 *70
* LED *70
* RGB LED *12
* TRRS線 *1
